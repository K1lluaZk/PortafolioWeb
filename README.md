### 👤 Sobre el Proyecto

Este repositorio contiene el código fuente de mi portafolio online. El objetivo de este proyecto es centralizar mi experiencia y formación en un entorno digital moderno, priorizando la experiencia de usuario (UX) y la optimización del rendimiento frontend.

### ✨ Características Técnicas

* **🎨 Diseño UI/UX Moderno:** Estética oscura con acentos en azul neón, utilizando tipografías modernas y una jerarquía visual clara.
* **📱 Arquitectura Mobile-First:** Completamente responsivo gracias al uso del sistema de rejilla de **Bootstrap 5**, adaptándose a móviles, tablets y monitores 4K.
* **✨ Animaciones de Interacción:** * **Reveal Effect:** Los elementos aparecen suavemente al hacer scroll utilizando `Intersection Observer API`.
* **Parallax Sutil:** Efecto dinámico en la imagen principal que reacciona al movimiento del mouse.
* **Smooth Scroll:** Navegación fluida entre secciones con compensación de altura para el menú persistente.


* **📊 Stack Tecnológico:** Sección dedicada que muestra mis herramientas principales mediante iconos vectoriales (SVG).
* **📁 Galería de Proyectos:** Maquetación de tarjetas con efectos de *overlay* para acceso directo a repositorios y demostraciones en vivo.

### 🛠️ Tecnologías Utilizadas

* **Frontend:** HTML5, CSS3, JavaScript (ES6+).
* **Framework CSS:** Bootstrap 5.
* **Iconografía:** Bootstrap Icons y SVG personalizados.
* **Animaciones:** JavaScript Nativo (sin librerías pesadas externas para mejorar la velocidad de carga).

### 🚀 Instalación y Despliegue Local

Para visualizar el portafolio en tu máquina local:

1. **Clonar el repositorio:**
```bash
git clone https://github.com/K1lluaZk/portafolio.git

```


2. **Navegar a la carpeta:**
```bash
cd portafolio

```


3. **Ejecutar:** Abrir `index.html` en cualquier navegador o usar la extensión **Live Server** en VS Code para ver cambios en tiempo real.
