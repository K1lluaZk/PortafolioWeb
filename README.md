# 📖 Acerca del Proyecto

**MarioWorld** es mi portafolio profesional como desarrollador de software, diseñado para mostrar mis conocimientos, proyectos y experiencia mediante una interfaz moderna, elegante y minimalista.

El sitio fue desarrollado utilizando **HTML, Tailwind CSS y JavaScript**, priorizando el rendimiento, la accesibilidad y una experiencia de usuario fluida.

La identidad visual está inspirada en un **cielo nocturno**, utilizando un fondo oscuro con estrellas animadas de forma sutil para transmitir calma, profesionalidad y un estilo tecnológico sin recurrir a efectos exagerados.

# 🚀 Instalación

### 1. Clonar el repositorio

```bash
git clone https://github.com/K1lluaZk/PortafolioWeb.git
```

### 2. Entrar al proyecto

```bash
cd PortafolioWeb
```

### 3. Instalar dependencias

```bash
npm install
```

### 4. Generar la hoja de estilos de Tailwind

```bash
npm run build
```

### 5. Ejecutar en modo desarrollo

```bash
npm run serve
```

# 🎨 Diseño

El diseño del portafolio se basa en una estética limpia y elegante, donde el contenido es el principal protagonista.

Entre sus características destacan:

- Fondo oscuro inspirado en el cielo nocturno.
- Estrellas con animaciones sutiles.
- Amplios espacios en blanco.
- Tipografía moderna.
- Componentes minimalistas.
- Transiciones suaves.
- Navegación intuitiva.

---

# 📬 Formulario de Contacto

El portafolio incorpora un formulario de contacto funcional utilizando **EmailJS**, permitiendo enviar mensajes directamente desde la página sin necesidad de desarrollar un backend dedicado.

Los mensajes enviados incluyen:

- Nombre del remitente.
- Correo electrónico.
- Asunto.
- Mensaje.
